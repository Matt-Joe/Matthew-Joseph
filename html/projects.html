Avanti Guest Lodge

This is a holiday booking website where users are able to book for and pay for their holidays. They can also create accounts to check their profile and bookings that they have made. The rest of the site consists of information about the lodge, the rooms available, and the services that they offer.

The site is built using HTML, CSS, and JavaScript for the front-end, Supabase for the database, Render for the back-end, and Stripe for payment processing. It was deployed on Netlify.

Intellecto

This is Learning Management System (LMS) where users are able to sign up as either a student or an instructor. Instructors are able to create courses, and students are able to enroll in those courses. Once enrolled, students can view the course content and track their progress.

The site is built using React for the front-end, Node.js and Express for the back-end, and Firebase for the database.

Civic Sync

This site is meant to bridge the gap in communication between residents and the municipality. User are able to create accounts, and once their account is created they are able to make a complaint or a request for an issue in their area. Admins will receive these and be able to accept or deny it, and if accepted, assign it to a contractor or anyone able to fix the issue.

The site is built using HTML, CSS, and JavaScript for the front-end, and Supabase for the database. It was deployed on Netlify.
